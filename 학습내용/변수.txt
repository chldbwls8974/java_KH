변수
학습일자: 0530~0531

* 요약*
- 자료형 : 8개
- 변수 선언 방법
- 변수명 작성 규칙 및 관례
- 자료형 변환

	[자료형] int, char, long, ... 총 8개

	[참조형 - String]
		String 클래스 : 문자열을 저장하는 클래스로 다양한 기능을 제공하는 클래스
		"" 를 이용하여 문자열을 관리하며 문자 0개 이상 가능하다.
		null 이용 가능 (참조변수라서)
	
			※ 빈 문자열과 null은 다르다!!
			>> 빈 문자열 : 화장실에 휴지가 있는데 휴지심만 있는 경우
			>> null : 화장실에 휴지 자체가 없는 경우

	
	[변수 명명 규칙]
		1. 대소문자 구분
		2. 예약어 사용 불가 // 예약어 :이미 자바에서 약속하여 사용하고 있는것
		3. 숫자로 시작 불가
		4. 특수문자는 _와 $만 가능 (공백도 불가능)
		5. 변수명 중복 선언 불가

		(관례)
		1. 변수명의 첫글자는 소문자로 시작 + 카멜표기법
		2. 상수는 대문자로 하며, 단어 구분은 _ 로 함.
		3. 클래스 명의 첫 글자는 대문자로 시작


	[변수, 상수, 리터럴]
		- 변수 : 하나의 값을 저장하는 공간
		- 상수 : 값을 저장 후 수정할 수 없는 공간 >> final이라는 키워드를 앞에 붙여줌으로써 생성 // ex) final int MAX_AGE = 100;
			때문에 선언과 동시에 초기화 하는것이 좋다.								※ static final에서는 선언과 동시에 반드시 초기화
		- 리터럴 : 그 자체로 값을 의미


	지역 변수는 사용되기 전 반드시 초기화 해야함. 


	[문자열]
		문자열 + 기본형 = 문자열			EX) "abc" + 1 + 2 = abc12
		문자열 + 문자열 = 문자열		 	     1 + 2 + "abc" = 3abc		

			
	[형변환]
		묵시적 : 작은 자료형 -> 큰 자료형
		명시적 : 큰 자료형 -> 작은 자료형  >> 데이터 손실 위험
